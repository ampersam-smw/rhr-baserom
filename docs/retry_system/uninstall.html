Uninstalling Retry

If you wish to uninstall the Retry System, and you're not using a tool like Callisto that can rebuild the rom from scratch easily, it is easy to do following these steps:

  • Delete the retry.asm file from UberASM Tool's library folder.
  • Delete all the retry_gmXX.asm files from UberASM Tool's gamemode folder.
  • Delete the retry_config folder from UberASM Tool's top folder.
  • In UberASM Tool's list.txt file, remove all the lines with retry_gmXX.asm under gamemode:.
  • Run UberASM Tool.
  • Patch the retry_unpatch.asm file found in the asm folder using Asar.
