User Toolbar

Lunar Magic has a built-in function to allow users to extend its toolbar with additional buttons (see the Lunar Magic help documentation if you need more details) and the baserom makes use of this to add some simple buttons that are some quick shortcuts to things.

NameShortcutDescription
Run ROM in EmulatorF4Opens the current ROM in configured Lunar Magic emulator.
Reload Current ROMCtrl+RReloads the currently open ROM.
Open Current ROM Directory--Opens the current project directory in File Explorer.
Open CallistoCtrl+BOpens Callisto in a new command-line window.
Run Update in CallistoCtrl+UUpdates your ROM by running "Update" in Callisto.
Make a BPS patch with CallistoCtrl+PCreates a BPS patch using Callisto's "Package" function.
Open Patches Directory--Opens the patches folder in the baserom's resources.
Edit Patches List--Open resources.toml so you can edit the list of patches.
Open PIXI Directory--Opens PIXI's directory in File Explorer.
Edit Sprite List--Opens PIXI's sprite list in default text editor.
Launch CFG Editor--Launches the sprite config editor tool that comes with PIXI.
Open GPS Directory--Opens GPS's directory in File Explorer.
Edit Custom Block List--Opens GPS's block list in default text editor.
Open UberASMTool Directory--Opens UberASMTool's directory in File Explorer.
Edit UberASM list--Opens the UberASM list in default text editor.
Open AddMusicK Directory--Opens AddMusicK's directory in File Explorer.
Edit Music List--Opens AddMusicK's custom music list in default text editor.
View Exit-Enabled TilesCtrl+F1Toggles the outlines on exit-enabled tiles in Lunar Magic.
View Tile OutlinesCtrl+F2Toggles the viewable tile outlines in Lunar Magic.
View Blue P-Switch/POW EffectsF12Displays the effects of a Blue P-Switch in the level view.
View Silver P-Switch/POW EffectsF11Displays the effects of a Silver P-Switch in the level view.
Toggle ON/OFF Switch Animation0Toggles any ON/OFF related animation.
Go Back to Previous Level--Goes back in your history of viewed levels.
Go Forward to Next Level--Goes forward in your history of viewed levels.
Open Baserom ReadmeCtrl+HOpens the baserom's included documentation.
User toolbar may differ between versions of the baserom.
